Git-gitlab

gitlab command line interface

Download this project as a .zip file Download this project as a tar.gz file

Overview

Git-gitlab is git's subcommand for Gitlab.You can operate some gitlab feature(so you don't use browser access).

Installation

Git-gitlab is managed by gem what is ruby's package management tool.So you are setting for use gem command, and then input below

gem install git-gitlab

Getting start

first, access to gitlab configure

git config --global gitlab.url http://gitlab.example.com/
git config --global gitlab.token GITLAB_SECRET_TOKEN

next. veryfi

git gitlab

Using

move to your local repository what is managed by gitlab.and

git config gitlab.project NAMESPACE/PROJECT

Get all openning issue

git gitlab issue

Get issue detail

git gitlab issue ISSUE_NUMBER

Get all openning mergerequest

git gitlab merge

and create mergerequest

git gitlab merge SOURCE_BRANCH TARGET_BRANCH --assign NAME

When you review mergerequest

git gitlab review MERGEREQUEST_ID

this command checkout your local repository to merged mergerequest.you can code review and run, debug, test.

Some configurations

you can configure project id

git config gitlab.projectid PROJECT_ID

when you use pproject id, then git-gitlab doesn't search project so access is more first.

If you use not gloabl gitlab, then move to local repository and

git config gitlab.url http://gitlab.example.com/
git config gitlab.token GITLAB_SECRET_TOKEN

you can access to specified gitlab
